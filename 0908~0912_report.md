## 자료형
 ```
 - 데이
 터를 저장하고 처리하는데 사용되는 기본 단위
 - type()로 자료형 확인 가능
 - 숫자형 : int, float
 - 문자열형 (Str) : 문자의 집합, ''나 ""로 감쌈
 순서(index)가 있어 슬라이싱, 반복, 탐색 가능
 - 불린형 (Bool) : 참 또는 거짓만을 가짐
 주로 조건문, 논리 연산에 사용
 - 리스트 (List) : 여러 값을 순서있게 저장할 수 있는 자료형
 리스트 내 요소 수정 가능
 []로 감쌈
 - 튜플 (Tuple) : 리스트와 비슷하지만 수정 불가능
 ()로 감쌈
 - 딕셔너리 (Dict) : key값과 value값 한 쌍으로 구성된 자료형
 {}로 감쌈
 - 집합 (Set) : 중복을 허용하지 않음
 순서가 없어 인덱싱 불가
 교집합, 합집합, 차집합 등 집합 연산 가능
{}로 감쌈
 - iterable : 하나씩 꺼낼 수 있는 것, 즉 반복할 수 있는 개체 (for문 등)
 - 문자열 (str), 리스트 (list), 튜플(tuple), 
 딕셔너리(dict), 집합(set), range 객체 등이 속함
 ```

## if 조건문

```
- if문

if 조건식 :  
    실행할 코드
- 조건식이 True면 코드 실행, False면 실행하지 않고 넘어감

- if - else문
if 조건식 :
    실행할 코드 1
else :
    실행할 코드 2

- 조건식이 True면 코드 1 실행
- False면 코드 2 실행

- if - elif - else문

if 조건식1 :
    실행할 코드 1
elif 조건식2 :
    실행할 코드 2
else :
    실행할 코드 3

- 여러 조건 중 하나만 실행하고 싶을 때
- 첫 번째 참인 조건의 코드만 실행 후 나머지 무시

```

## 리스트 내 인덱싱


## 함수

### 자료형 변환 함수
```
- int()
int("123") --> 123
- float()
float("3.14") --> 3.14
- str()
str(100) --> "100"
- list()
반복 가능한 객체(iterable)를 리스트로 변환
list("abc") --> ['a', 'b', 'c']
- tuple()
반복 가능한 객체(iterable)를 리스트로 변환
tuple([1, 2, 3]) --> (1, 2, 3)
```
